Setup Instructions

Prerequisites

  • Python 3.11 or higher
  • OpenAI API key
  • Poetry installed (see: https://python-poetry.org/docs/#installation)

Installation

1. Download and extract the template

Download the research paper summarizer template ZIP file and extract it to your desired location:


unzip research-paper-summarizer-agent.zip
cd research-paper-summarizer-agent

2. Install dependencies


poetry install

This will automatically install GraphBit and all other required dependencies listed in pyproject.toml.

3. Set up environment variables


export OPENAI_API_KEY="your_api_key_here"

Running the Application

Using Poetry

Step 1: Start the backend server

In terminal 1:

poetry run uvicorn backend.main-server:app --reload --host 0.0.0.0 --port 8000

Step 2: Start the frontend application

In terminal 2:

poetry run streamlit run frontend/app.py --server.port 8501 --server.address 0.0.0.0

Note: The startup scripts require GraphBit to be properly installed and available in your Python environment.

Access the Application

  • Frontend: http://localhost:8501
  • Backend API: http://localhost:8000
  • API Documentation: http://localhost:8000/docs
